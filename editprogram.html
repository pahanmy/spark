Ralat Kebenaran Akses Pangkalan Data

Tindakan ini gagal kerana tetapan keselamatan (Security Rules) pada pangkalan data Firebase anda tidak membenarkannya. Anda MESTI mengemas kini fail firestore.rules bagi membenarkan akses `update` kepada Penyelaras program (UID). 

match /artifacts/{appId}/public/data/spark_programs/{programId} {
  allow read: if true;
  // Ubah baris 'allow update' ini:
  allow update: if request.auth.uid == resource.data.penyelaras_uid;
  allow delete: if request.auth.uid == resource.data.penyelaras_uid;
}
